A HTML doc starts always with a '!DOCTYPE html' element, and then we have a 'html' + definition of language to be used in the doc.
After that we have the 'head' element, which will hold the title (necessary), stylesheets, and scripts to run in your webpage.
After we have the 'body' element, that will hold all the parts to be displayed in your webpage.

Starting with the definition of markup:
A markup is the way of formatting the appearence or use of your elements on your browzer.

Elements

Examples:
An 'em' inside angle brackets is made to italicize something
A 'strong' inside angle brackets is used to make the word boldface
'Br' inside angle brackets
breaks
the line.
'Br' is a void element: Does not need a end tag.

Let's try a few other HTML elements:
This one it called 'mark'.
Another one is 'sub' and 'sup'.
We can use more than one type of tags in our text at once.
Like 'em and mark', be careful nesting them correctly.

An element can be a block or inline element.
A block element create an invisible box with width, height and margin before and after.

An element is made of tag(s) and content.
Usually, we have an opening tag, the content and a closing tag.
Like this: opening tag + content + closing tag
For some elements the closing tags are optional, like 'li'(list) and 'p'(paragraph).
This happen because the browser knows there has to be a closing tag before the next opening tag for the same element.
There is no paragraph inside a paragraph.
Some elements does not take any closing tag, like 'br'.

A division element separate what is inside it in a block, as well as a paragraph element. It also creates an invisible box.

Lists

The 'li' element stands for list. A list can be ordered - 'ol' or unordered - 'ul'
An unordered list has bullets (default):

  • Books
  • Pencil
  • Pens
  • Notebook
An ordered list has numbers (default):
  1. Books
  2. Pencil
  3. Pens
  4. Notebook
We can create a list inside a list, it is called nested list, like this:
  • Animals
    1. Dog
    2. Horse
    3. Cat
  • Flowers
    1. Sunflower
    2. Roses

Image

To add some image to your website it will be needed to add an 'img' element.
The 'img' element takes a source 'src' and a 'alt' references.
The 'src' reference is the address of the image, and the 'alt' is what will be showed if the image can't be loaded.
